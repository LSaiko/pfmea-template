Quick start

From clone to a three-format pFMEA in four lines.

git clone https://github.com/LSaiko/pfmea-template.git && cd pfmea-template
pip install -r requirements.txt

python pfmea.py list                                          # see scenarios
python pfmea.py generate sterile_packaging --format all --iso14971
# -> pfmea_sterile_packaging.xlsx / .docx / .pdf  (+ ISO 14971 bridge sheet)

python pfmea.py changelog --from old.yaml --to scenarios/cnc_femoral_stem.yaml
